# stock-management-system-swing

Desktop application for managing grocery shop inventory and sales.

Built with Java Swing and connected to a MySQL database using JDBC.

## Features

• Add new products  
• Update product stock  
• Record product sales  
• View current inventory  

## Technologies

Java  
Java Swing  
JDBC  
MySQL  

## System Architecture

Java Swing Desktop App → JDBC → MySQL Database

## Database Setup

1. Install MySQL or XAMPP
2. Open phpMyAdmin
3. Run the SQL script in:

database/magasindb.sql

## Running the Application

Compile and run the Java files:

javac Main.java  
java Main
